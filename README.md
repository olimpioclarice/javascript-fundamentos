# JavaScript Fundamentos

Este projeto foi desenvolvido como exercício de introdução ao JavaScript.

## Objetivo

Demonstrar a execução de scripts no navegador utilizando:

- `alert()`
- `document.write()`
- Scripts no `<head>` e no `<body>`

## Tecnologias

- HTML5
- JavaScript

## Como executar

1. Baixe o projeto.
2. Abra o arquivo `index.html` em qualquer navegador.

## Resultado esperado

Ao abrir a página:

- Um alerta é exibido na tela.
- Após fechar o alerta, uma mensagem em negrito é escrita na página.
